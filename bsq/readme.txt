Дано:

Файл с картой:
	число n и стри символа
	n строк из этих символов

Ошибки:
	++*неправильная первая строка(последние три символа - заполнение карты, остальные - число)
	++разная длина строк
	++ни одной строки или ни одного симва
	++испльзованы символы не из первой строки
	++не хватает символов в первой строке или они одинаковые
	++непечатаемые символы

В случае ошибки:
	map error

Файлов с картой может быть много!!
После каждого файла первод строки и идём дальше

#
#Если файлов нет, считываем построчно из консоли, как в cat
#

Решение:
	если несколько решений, берём левое верхнее

Нужен Malefile
